# Wx

This weather gem gives you real-time weather information from any city in the world. Simply type the city name, or any US zip code. 

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'wx'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install wx
